== Description ==

This plugin allows the customer to "Like" on Facebook about your store during Checkout & earn a discount. This plugin can lead to **exponential growth in sales** as each & every customer is given an incentive to promote your store.

== Installation ==

1. Ensure you have latest version of WP e-Commerce plugin installed
2. Unzip and upload contents of the plugin to your /wp-content/plugins/ directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. The "Viral Coupon" link will appear in the admin panel once the plugin is activated.

[Click here](http://www.tychesoftwares.com/manuals/How-to%20Manual%20For%20Viral%20Coupon%20Plugin%20Lite.pdf "How-to Manual For Viral Coupon Plugin Lite") to download the How-to manual for the plugin.

== Frequently Asked Questions ==

= I have installed & activated the plugin, why is the sharing box not visible on the Checkout page? =

Once installed & activated, please go to the "Viral Coupon" settings page & enter all the configuration parameters. Once completed, you will see the sharing box on the checkout page.
